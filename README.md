# user-typing-speed-
a program that allows user to test how quickly and accurately they can type a given set of characters 
----
The program generates a list of random charcaters
minimum - 5
maximum - 15

---
The generated displayed charcters is then inputted by the user to calculate thier typing speed 
time taken starts when usersbegins typing 
stop timing when they finish 
total time taken is calculated 
---
A Validation Rule
Name should not be empty 
contain only letters 
numbers of character must be between 5 and 15 
---
Overall Goal 
test typing speed(Time taken)
test typing accuracy 
handle user input safely using validation
